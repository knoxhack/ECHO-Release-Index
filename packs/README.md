# Pack Manifests

Generated pack manifests for Ashfall Native Edition, Ashfall NeoForge Edition, and Ashfall Standalone Edition will be committed here after release assets are uploaded.

Each edition pins module artifacts from `knoxhack/ECHO-Modules`:

| Pack | Artifact family |
| --- | --- |
| `ashfall-native-edition.json` | `<module>-<version>.echo-addon` |
| `ashfall-neoforge-edition.json` | `<module>-<version>-neoforge.jar` |
| `ashfall-standalone-edition.json` | `<module>-<version>-standalone.jar` |
